# Unifacef React Native Course Repository

## Description

Repositório do curso de React Native da pós da Unifacef.

## Installation

1.  Clone the repository: `git clone https://github.com/unifacef/unifacef-react-native.git`
2.  Navigate to the repository: `cd unifacef-react-native`
3.  Install dependencies: `npm install` (or `yarn install` if using Yarn)

## Usage

This repository contains the materials for a React Native course taught by the Unifacef faculty.

*   **`aula-1`**: Introduction to React Native.
*   **`aula-2`**: Building UI Components.
*   **`aula-3`**: State Management.
*   **`aula-4`**: Navigation and Routing.
*   **`contato.md`**:  Contact Information.
*   **`images`**:  Screenshots and visual assets.
*   **`readme.md`**: Course Description & Setup Instructions.
*   **`repositorios.md`**:  Repository Structure Details.
*   **`requisitos.md`**:  Course Prerequisites.

**Key Resources:**

*   `README.md`: This file contains the overall project description and setup instructions.
*   `aula-1`, `aula-2`, `aula-3`, `aula-4`:  Contains the lecture materials for each module.
